# 2048 Game

A modern, sleek implementation of the classic 2048 puzzle game with a stunning dark theme UI and smooth animations.

## ğŸ® About The Game

2048 is an addictive sliding block puzzle game where the objective is to combine numbered tiles to create a tile with the number 2048. Players slide tiles in four directions (up, down, left, right), and matching tiles merge into one with their sum. The game continues until the player achieves the 2048 tile or no more moves are possible.
